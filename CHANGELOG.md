# Changelog

## 0.1.0

- Initial release
- DNG file preview with full raw sensor decode (libraw-wasm)
- Fast-path embedded JPEG thumbnail extraction (exifr)
- Zoom/pan controls (mouse wheel, drag, keyboard shortcuts)
- EXIF metadata sidebar
- Theme-aware UI (dark, light, high-contrast)
- Configurable half-size decode and white balance settings
